## 4. Frontend — capture pipeline: Audio capture pipeline (Decision 2) + WebSocket reconnect strategy (Decision 3)

- [x] 4.1 [P] Implement Decision 2: Audio capture pipeline — AudioWorklet + manual 48→16 kHz downsampling at `frontend/src/capture/audio-worklet.ts` (linear interpolation when source rate ≠ 16 kHz, passthrough when rate = 16 kHz) feeding the `Real-time captioning UI consumes WS /listen` capability.
