## 1. Postprocess module + tests (foundation)

- [x] 1.1 Create `app/services/postprocess.py` exporting `Keep`, `Drop`, `FilterDecision`, and `filter_empty_transcription(text, duration_ms, *, enabled, min_duration_ms)` per the design Implementation Contract section 1. The punctuation set SHALL be documented inline. The function is a pure function (no side effects, no logging — callers log). Verification: new `tests/test_postprocess.py` parameterised over enabled-bypass, sub-duration drop, empty-text drop, whitespace-only drop, ASCII-punctuation-only drop, CJK-punctuation-only drop (`。`, `，`, `、`, `；`, `：`, `？`, `！`, `「」`, `『』`, `（）`, `《》`, `〈〉`, `…`, `—`, `·`), mixed punctuation+space drop, CJK single-char "好" Keep, English "Hi." ## 5. /ask integration

- [x] 5.1 Integrate the filter into `app/api/ask.py` for the audio paths (multipart, raw `audio/*`, octet-stream) per the design Implementation Contract section 5. Behavior: after STT for audio inputs, call `filter_empty_transcription(text=transcript, duration_ms=None, ...)`. On `Drop` for the blocking path, return `JSONResponse({"error": "no_speech_detected"}, status_code=400)` AND log `"transcription_filtered"` at INFO with `extra={"endpoint": "/ask", "reason": <decision.reason>, "stream": false, "raw_text_len": <int>}`. On `Drop` for the SSE path (`?stream=true`), emit exactly one frame `event: error\ndata: {"error": "no_speech_detected"}\n\n` then close cleanly; the LLM client SHALL NOT be invoked; log line gets `extra.stream=true`. JSON text-input path SHALL be untouched (no filter call there). When `FILTER_EMPTY_ENABLED=false`, the audio path SHALL behave as before (no 400 from filtering). Verification: extend `tests/test_ask.py` with: multipart audio mock-transcribing to `"。"` returns 400 + LLM mock asserted not called via `assert mock_llm.generate.call_count == 0`; SSE same input emits one `event: error` frame + LLM not called; JSON text input `{"text": "hello"}` unaffected (LLM IS called); `FILTER_EMPTY_ENABLED=false` audio empty case calls LLM as legacy. `uv run pytest tests/test_ask.py -v` SHALL pass.
